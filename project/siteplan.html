Website Project Site Plan

Prepared by: Winfred Wambui

Site Name

Global Temples Explorer – This name reflects the purpose of the website: to showcase different temples around the world with history, images, and visitor information.

Optional domain: globaltemples.org

Site Purpose

The site provides historical and cultural information about significant temples worldwide. Visitors can learn about each temple’s background, see images, and find travel details.

Scenarios

  • Which temples are most popular to visit in Europe and Africa?
  • Where can I find visitor information and travel details for a specific temple?

Color Schema

Primary Color: #2C3E50 (used for headers and navigation background)

Accent Color: #E67E22 (used for highlights, buttons, and section headings)

This document also uses these colors to reflect the chosen schema.

Typography

Heading Font: "Merriweather", serif – for main headings.

Body Font: "Open Sans", sans-serif – for paragraphs and general text.

By Winfred Mercy Wambui

Site Name

SafeNet Security Hub — This site is focused on providing cybersecurity tips, awareness, and resources. The name emphasizes safety and network protection, aligning with the project’s goal of creating awareness in cybersecurity.

Site Purpose

The website aims to educate users about common cybersecurity threats and how to stay safe online. It will provide guides, tools, and updates on the latest security best practices while serving as a hub for cybersecurity awareness.

Scenarios

  • How can I recognize a phishing email?
  • What steps should I take to protect my personal data online?

Color Schema

Selected colors:

  • #1E3A8A (Navy Blue) — for headers and navigation background.
  • #FACC15 (Amber) — for highlights, buttons, and accents.

Typography

Selected fonts:

  • Playfair Display — for headings (h1, h2, h3).
  • Roboto — for body text, lists, and descriptions.
